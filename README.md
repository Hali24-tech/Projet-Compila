# Pseudo-Code Analyzer

## Description
Ce projet implémente un mini-compilateur de pseudo-code avec :
- analyse lexicale
- analyse syntaxique
- analyse sémantique

Les différents modules sont intégrés dans une seule application
accessible via une interface en ligne de commande.

## Utilisation

### Compilation
```bash
gcc analyseurlexical.c analyseursyntaxique.c -o analyzer
